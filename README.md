# Freetime Maker Shop

The official online shop from Freetime Maker. Here you can buy and download games and apps for various platforms like Android, Windows, and Linux.

## 📋 Description

This website is a simple HTML-based shop system that enables the purchase and download of games and applications. Payments are processed through various providers, including cryptocurrencies via Coinbase. After purchase, you will receive a code via email to unlock the product.

## 💳 Purchase Process

1. Select a product and click "Pay via Different Providers".
2. Provide your email address.
3. Complete the payment (supports various payment methods, including Coinbase for cryptocurrencies).
4. After payment confirmation, you will receive a product code via email.
5. You will also be notified via email when updates are available.

**Note:** Cash payments are currently not available. For questions about payment methods in Switzerland or for teenagers, create a ticket on the Discord server.

## 🚀 How to Host the Website

This website is static and consists of HTML, CSS, and image files. To host it:

1. Clone the repository:
   ```bash
   git clone https://github.com/FreetimeMaker/Freetime-Maker-Shop.git
   cd Freetime-Maker-Shop
   ```

2. Open `index.html` in a web browser or host the files on a web server (e.g., Apache, Nginx, or GitHub Pages).

No additional dependencies required.
